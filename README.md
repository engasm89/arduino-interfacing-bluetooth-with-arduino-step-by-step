## Overview

Control the Arduino onboard LED via Bluetooth commands using an HC-05 module. Commands like `LED ON` and `LED OFF` are received and executed.

## Code Explained

- Initializes serial communication at 9600 baud.
- Buffers incoming text and executes commands on newline.
- Manipulates the `D13` LED as feedback.

## How To Use

- Pair your phone with the HC-05 and use a Bluetooth terminal app.
- Send `LED ON` or `LED OFF` to control the LED.
